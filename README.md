# adamant-docs

Developer documentation for Adamant Protocol.

Guides, tutorials, integration examples, and API references for developers building on Adamant.

## Status

Pre-alpha. Forthcoming alongside testnet.

## What this is

A developer documentation site, separate from the public marketing website at [adamant-website](https://github.com/adamant-protocol/adamant-website). Where the website orients newcomers, this repository serves people actually building software that uses Adamant.

Includes:

- Getting-started guides for SDK users
- Wallet integration tutorials
- Contract deployment walkthroughs
- API references for the SDKs and node RPC
- Best practices for privacy-preserving applications

## What this is not

Not the formal specification — see [adamant-spec](https://github.com/adamant-protocol/adamant-spec) for the protocol's authoritative description.

Not the public website — see [adamant-website](https://github.com/adamant-protocol/adamant-website) for the marketing-facing site.
